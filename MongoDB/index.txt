Q: Students in Physics

  { name: "Eve", age: 20, course: "Physics", grades: [95] }

Query:

{ acknowledged: true, matchedCount: 1, modifiedCount: 1 }

5. Delete Charlie
Query:

8. Age 21 or Course = Chemistry

[
  { name: "Alice", ... },
  { name: "Charlie", ... }

9. Group by Course
Query:

  { $group: { _id: "$course", count: { $sum: 1 } } }
])
Output:
[
  { _id: "Math", count: 2 },
  { _id: "Physics", count: 2 },
  { _id: "Chemistry", count: 1 }

10. Sort by Age Desc
Query:
db.students.aggregate([{ $sort: { age: -1 } }])
Output:
[
  { name: "David", age: 24, ... },
  { name: "Bob", age: 23, ... },
  ...
]

11. Project Name + Age
Query:
db.students.aggregate([
  { $project: { name: 1, age: 1, _id: 0 } }
])
Output:
[
  { name: "Alice", age: 21 },
  { name: "Bob", age: 23 },
  ...
]

12. Avg Grade per Student
Query:
db.students.aggregate([
  { $project: { name: 1, avgGrade: { $avg: "$grades" } } }
])
Output:
[
  { name: "Alice", avgGrade: 86.33 },
  { name: "Bob", avgGrade: 80 },
  ...
]
